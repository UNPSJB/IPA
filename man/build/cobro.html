Canon Infraccion¶

Comó crear un nuevo cobro de infraccion¶

Desde el listado de permisos seleccionamos el Permiso al cual le vamos a realizar el Cobro de infraccion.

Nos dirigimos al icono Mas Acciones el cual nos desplegara opciones, seleccionamos Nuevo Cobro de Infraccion.

Para terminar registrar el Cobro de infraccion debemos completar Descripcion, Archivo, Fecha del documento, Monto del Cobro.

Comó crear un nuevo pago de infraccion¶

Desde el listado de permisos seleccionamos el Permiso al cual le vamos a realizar el Pago de infraccion.

Nos dirigimos al icono Mas Acciones el cual nos desplegara opciones, seleccionamos Nuevo Pago de Infraccion.

Para terminar registrar el Pago de infraccion debemos completar Descripcion, Archivo, Fecha del documento, Monto del Cobro.
